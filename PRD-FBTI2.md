## 五、FBTI 2.0 球迷人格测试详细设计（Football Behavior Type Indicator）

### 5.1 架构概述

本测试采用**5大模型 × 15维度**的球迷场景化人格测试框架，深度融合球迷行为特征与网感命名风格，命名为 **FBTI 2.0（Football Behavior Type Indicator / 球迷行为类型指标）**。

参考SBTI的向量匹配算法，但全部维度围绕球迷场景重新设计，避免抄袭感。

**5大模型定义**：

| 模型 | 英文 | 维度 | 球迷场景解释 |
|------|------|------|-------------|
| **观赛方式** | Watching (W) | W1 观看场景 / W2 投入程度 / W3 时间偏好 | 在哪里看、怎么看、什么时候看 |
| **情感模式** | Emotion (E) | E1 主队绑定 / E2 情绪反应 / E3 胜负观 | 爱得多深、反应多大、输赢怎么看 |
| **知识储备** | Knowledge (K) | K1 规则理解 / K2 球员认知 / K3 战术水平 | 懂不懂球、认不认识人、看不看得懂战术 |
| **社交行为** | Social (S) | S1 表达方式 / S2 互动风格 / S3 身份展示 | 怎么表达、怎么互动、怎么展示 |
| **消费决策** | Money (M) | M1 投入意愿 / M2 消费方向 / M3 价值判断 | 愿不愿花钱、花在哪、值不值 |

**测试规则**：
- **总题数**：30题（15维度 × 2题）+ 2道隐藏触发题
- **每题选项**：3个选项（value 1/2/3），分别对应低/中/高倾向
- **计分方式**：每维度原始分 = 2题之和（范围2-6），映射为 L(低≤3) / M(中=4) / H(高≥5)
- **匹配算法**：15维用户向量 vs 25个人格模板，曼哈顿距离最近邻匹配
- **相似度**：similarity = max(0, round((1 - distance/30) × 100))
- **特殊机制**：
  - 隐藏人格DRUNK：特定题目触发，直接判定
  - 兜底人格HHHH：相似度<60%时触发

### 5.4 评分算法

```javascript
// 1. 计算15维度原始分（每个维度2题，每题1-3分，范围2-6）
const dimensionScores = {
  W1: q1 + q2, W2: q3 + q4, W3: q5 + q6,
  E1: q7 + q8, E2: q9 + q10, E3: q11 + q12,
  K1: q13 + q14, K2: q15 + q16, K3: q17 + q18,
  S1: q19 + q20, S2: q21 + q22, S3: q23 + q24,
  M1: q25 + q26, M2: q27 + q28, M3: q29 + q30
};

// 2. 映射为 L/M/H
const level = score => score <= 3 ? 'L' : score === 4 ? 'M' : 'H';

// 3. 生成用户向量（L=1, M=2, H=3）
const userVector = Object.values(dimensionScores)
  .map(level)
  .map(l => ({L:1, M:2, H:3})[l]);

// 4. 计算与25个人格模板的曼哈顿距离
const templates = [
  { name: 'CTRL', pattern: 'HHH-HHH-HHH-HHH-HHH' },
  // ... 其他24个模板
];

const results = templates.map(t => {
  const tVector = t.pattern.split('-').join('').split('')
    .map(c => ({L:1, M:2, H:3})[c]);
  const distance = userVector.reduce((sum, v, i) => 
    sum + Math.abs(v - tVector[i]), 0);
  const exact = userVector.filter((v, i) => v === tVector[i]).length;
  const similarity = Math.max(0, Math.round((1 - distance / 30) * 100));
  return { ...t, distance, exact, similarity };
}).sort((a, b) => 
  a.distance - b.distance || 
  b.exact - a.exact || 
  b.similarity - a.similarity
);

// 5. 特殊判定
if (drunkTrigger) return { type: 'DRUNK', similarity: 100 };
if (results[0].similarity < 60) return { type: 'HHHH', similarity: results[0].similarity };
return results[0];
```

### 5.6 分享海报设计

**海报布局**：
- 顶部：FBTI 2.0 Logo + "测测你是哪种球迷"
- 中间：人格代码（大字体）+ 中文名 + 英文名
- 徽章图标：对应人格的emoji/图标
- 颜色主题：根据人格类型变换背景色
- 描述文案：一句话概括人格特征
- 装懂金句：精选一条展示
- 底部：小程序码 + "扫码测测你的球迷人格"

**分享文案模板**：
```
测出我是「{中文名}」({代码})！
{装懂金句}
快来测测你是哪种球迷！
```

**示例**：
- CTRL："测出我是「CTRL 拿捏者」！根据我整理的近5年数据... 快来测测你是哪种球迷！"
- DRUNK："测出我是「DRUNK 酒鬼」！看球不喝酒，等于没看球！ 快来测测你是哪种球迷！"
