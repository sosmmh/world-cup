## FBTI 2.0 重大升级：V2 → V3 完整重构

### 升级概览
将 MBTI 球迷测试从 **V2（10种结果/随机15题）** 全面升级为 **FBTI 2.0（27种人格/固定32题+向量匹配+雷达图）**。

### 核心变化

| 维度 | V2（旧版） | V3 FBTI 2.0（新版） |
|------|------------|---------------------|
| 题库 | 40题池随机抽15题 | **32题**固定全作答（30常规+2隐藏彩蛋） |
| 选项 | 4选项(A/B/C/D) 0~3分 | **3选项**(A/B/C) 1~3分 |
| 计分 | 百分比分档 | **5模型×15维向量 + 曼哈顿距离最近邻匹配** |
| 结果数 | **10种** | **27种**（25常规+1隐藏DRUNK+1兜底HHHH） |
| 展示 | 中文名+百分比 | **4字母代码(Ctrl/GOAT)+中文名+相似度%+雷达图** |
| 彩蛋 | 无 | **DRUNK酒鬼**隐藏人格（H1选C且H2选C触发） |
| 视觉 | 绿色球场主题 | **深色科技主题(#0d1b2a)**+Canvas雷达图 |

## FBTI 结果页 V4 视觉升级（深夜）

### 问题
用户反馈背景太白、整体不好看，缺乏设计感。

### 改动
**`mbti-result.wxss` 全面重写为 V4 深色精致版：**
- **页面背景**：从淡白渐变改为深邃星空渐变 `#0f0c29 → #1a1a3e → #24243e → #16132b`，营造足球场夜景氛围
- **卡片样式**：毛玻璃半透明 `rgba(255,255,255,0.08)` + 微光金色顶边装饰线 + 多层阴影 + 内发光边框
- **文字颜色**：全部适配深色背景，主文字 `rgba(255,255,255,0.82~0.92)`，次要文字半透明白
- **相似度数字**：加金色辉光 `text-shadow: 0 0 30rpx rgba(255,215,0,0.4)`
- **分享按钮**：金→橙渐变 + 金色投影；复制按钮改为半透明玻璃质感
- **金句卡片**：暖色半透明渐变背景 + 金边

**`mbti-result.js` Canvas 绘制适配深色主题：**
- Canvas 添加深色半透明底 `rgba(0,0,0,0.2)` + 圆角
- 维度名称改为白色 `rgba(255,255,255,0.88)`
- 进度条轨道改为 `rgba(255,255,255,0.08)` 暗轨
- "低"等级药丸颜色从 `#90A4AE` 调暗为 `#78909C`
- 小标题改为淡白 `rgba(255,255,255,0.45)`
- 动画名从 popIn 改为 cardFloatIn，更柔和
