# 世界杯装懂指南 - 产品需求文档 (PRD)

## 一、项目概述

### 1.1 产品名称
世界杯装懂指南

### 1.2 产品定位
面向"四年一度球迷"的世界杯主题微信小程序，帮助不太懂球但不想被看出来的用户，通过趣味内容、测试和工具快速融入球迷圈子，假装资深老球迷。

### 1.3 目标用户
- 世界杯期间跟风看球的"伪球迷"
- 想参与社交话题但缺乏足球知识的新手
- 希望快速获取谈资、融入球迷圈子的用户

### 1.4 核心目标
- 趣味介绍2026美加墨世界杯48强球队及球员
- 提供球迷版MBTI测试，增强社交分享属性
- 提供多种"装懂"工具，降低新手门槛
- 实时获取足坛热点、赛程、赛果、新闻，保持内容新鲜度

## 二、数据获取策略总览

> **核心原则**：所有动态变化的数据（热点、赛程、赛果、新闻、实时比分）必须通过API实时获取；所有静态知识内容（球队百科、球员资料、黑话词典、测试题目等）使用本地数据或云数据库。

| 数据类型 | 获取方式 | 说明 |
|---------|---------|------|
| 足坛热点/新闻 | **实时API** | 必须实时获取，不可用本地数据 |
| 赛程/赛果/积分榜 | **实时API** | 必须实时获取，不可用本地数据 |
| 实时比分 | **实时API** | 必须实时获取，不可用本地数据 |
| 球队百科（48强） | 本地数据 | 静态知识，写入本地JS文件 |
| 球员详细资料 | 本地数据 | 静态知识，写入本地JS文件 |

### 3.5 球迷MBTI测试 (pages/mbti/mbti)

#### 功能描述
通过30道抽象幽默的场景化题目+2道隐藏触发题，基于**FBTI 2.0五模型十五维度**框架，测试用户的"球迷人格类型"，结果具有强烈的社交分享属性。

#### 核心功能
- **30道常规测试题**：每题3个选项（value 1/2/3分），覆盖5大模型15个维度（本地数据）
- **2道隐藏触发题**：特定选项组合触发隐藏人格"DRUNK 酒鬼"
- **十五维度计分**：每维度2题，原始分2-6，映射为L(低≤3)/M(中=4)/H(高≥5)
- **向量匹配算法**：15维用户向量 vs 25个人格模板，曼哈顿距离最近邻匹配
- **27种结果**：25种常规人格 + 1种隐藏人格(DRUNK) + 1种兜底人格(HHHH)
- **结果跳转**：测试完成后跳转至结果页

#### 计分规则
- 每道题选A/B/C，分别对应value 1/2/3分
- 每个维度2道题，原始分 = 两题value之和（范围2-6）
- 维度等级映射：score ≤ 3 → L(低)，score = 4 → M(中)，score ≥ 5 → H(高)
- 生成15维用户向量（如 H-L-M-H-H-L-M-M-H-L-L-H-M-M-H）
- 与25个人格模板的15维Pattern计算曼哈顿距离，取最近邻
- 相似度 similarity = max(0, round((1 - distance/30) × 100))
- **特殊判定**：
  - 隐藏题H1选C且H2选C → 直接判定为 DRUNK（相似度100%）
  - 最高相似度 < 60% → 判定为 HHHH（标准球迷）

## 五、FBTI 2.0 球迷人格测试详细设计（Football Behavior Type Indicator）

### 5.1 架构概述

本测试采用**5大模型 × 15维度**的球迷场景化人格测试框架，深度融合球迷行为特征与网感命名风格，命名为 **FBTI 2.0（Football Behavior Type Indicator / 球迷行为类型指标）**。

参考SBTI的向量匹配算法，但全部维度围绕球迷场景重新设计，避免抄袭感。

**5大模型定义**：

| 模型 | 英文 | 维度 | 球迷场景解释 |
|------|------|------|-------------|
| **观赛方式** | Watching (W) | W1 观看场景 / W2 投入程度 / W3 时间偏好 | 在哪里看、怎么看、什么时候看 |
| **情感模式** | Emotion (E) | E1 主队绑定 / E2 情绪反应 / E3 胜负观 | 爱得多深、反应多大、输赢怎么看 |
| **知识储备** | Knowledge (K) | K1 规则理解 / K2 球员认知 / K3 战术水平 | 懂不懂球、认不认识人、看不看得懂战术 |
| **社交行为** | Social (S) | S1 表达方式 / S2 互动风格 / S3 身份展示 | 怎么表达、怎么互动、怎么展示 |
| **消费决策** | Money (M) | M1 投入意愿 / M2 消费方向 / M3 价值判断 | 愿不愿花钱、花在哪、值不值 |

**测试规则**：
- **总题数**：30题（15维度 × 2题）+ 2道隐藏触发题
- **每题选项**：3个选项（value 1/2/3），分别对应低/中/高倾向
- **计分方式**：每维度原始分 = 2题之和（范围2-6），映射为 L(低≤3) / M(中=4) / H(高≥5)
- **匹配算法**：15维用户向量 vs 25个人格模板，曼哈顿距离最近邻匹配
- **相似度**：similarity = max(0, round((1 - distance/30) × 100))
- **特殊机制**：
  - 隐藏人格DRUNK：特定题目触发，直接判定
  - 兜底人格HHHH：相似度<60%时触发

### 5.4 评分算法

```javascript
// 1. 计算15维度原始分（每个维度2题，每题1-3分，范围2-6）
const dimensionScores = {
  W1: q1 + q2, W2: q3 + q4, W3: q5 + q6,
  E1: q7 + q8, E2: q9 + q10, E3: q11 + q12,
  K1: q13 + q14, K2: q15 + q16, K3: q17 + q18,
  S1: q19 + q20, S2: q21 + q22, S3: q23 + q24,
  M1: q25 + q26, M2: q27 + q28, M3: q29 + q30
};

// 2. 映射为 L/M/H
const level = score => score <= 3 ? 'L' : score === 4 ? 'M' : 'H';

// 3. 生成用户向量（L=1, M=2, H=3）
const userVector = Object.values(dimensionScores)
  .map(level)
  .map(l => ({L:1, M:2, H:3})[l]);

// 4. 计算与25个人格模板的曼哈顿距离
const templates = [
  { name: 'CTRL', pattern: 'HHH-HHH-HHH-HHH-HHH' },
  // ... 其他24个模板
];

const results = templates.map(t => {
  const tVector = t.pattern.split('-').join('').split('')
    .map(c => ({L:1, M:2, H:3})[c]);
  const distance = userVector.reduce((sum, v, i) => 
    sum + Math.abs(v - tVector[i]), 0);
  const exact = userVector.filter((v, i) => v === tVector[i]).length;
  const similarity = Math.max(0, Math.round((1 - distance / 30) * 100));
  return { ...t, distance, exact, similarity };
}).sort((a, b) => 
  a.distance - b.distance || 
  b.exact - a.exact || 
  b.similarity - a.similarity
);

// 5. 特殊判定
if (drunkTrigger) return { type: 'DRUNK', similarity: 100 };
if (results[0].similarity < 60) return { type: 'HHHH', similarity: results[0].similarity };
return results[0];
```

### 12.2 越位1分钟看懂 (type: offside)

**核心规则**：进攻球员在队友传球的瞬间，如果比球和倒数第二名防守球员更靠近对方球门，就处于越位位置。

**4种不越位情况**：

| 编号 | 情况 | 说明 | 装懂话术 |
|------|------|------|----------|
| 1 | 本方半场不越位 | 无论队友在哪里传球，只要接球时你在自己半场，就不越位 | "只要在本方半场接球，永远不会越位" |
| 2 | 回传球不越位 | 如果队友往你的方向回传（即球的运动方向是朝向你的后场），不越位 | "回传球不存在越位，这是基本常识" |
| 3 | 球比你先出去不越位 | 如果传球时球比你的位置更靠近对方底线，不越位 | "球比人先出去，这球不越位" |
| 4 | 角球/球门球/界外球不越位 | 这三种情况下不存在越位规则 | "角球、球门球、界外球没有越位一说" |

**装懂金句**："越位看的是传球瞬间，不是接球瞬间，VAR画线就是这个原理。"
